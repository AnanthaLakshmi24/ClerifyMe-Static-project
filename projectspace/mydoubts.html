Answered Questions

What is the difference between var, let, and const in JavaScript?

Answer: var is function-scoped and can be re-declared. let and const are block-scoped; let can be reassigned, while const cannot.

Explain the concept of React Hooks.

Answer: React Hooks allow function components to use state and lifecycle methods. Common hooks include useState, useEffect, and useContext.
